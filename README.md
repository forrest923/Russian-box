# 🎮 俄罗斯方块 - Russian Box

一个使用纯 HTML5 Canvas + JavaScript 开发的俄罗斯方块游戏。

## 🌐 在线试玩

👉 [点击这里开始游戏](https://forrest923.github.io/Russian-box/)

## 🎯 游戏特性

- ✅ 经典的俄罗斯方块玩法
- ✅ 7 种不同形状的方块 (I, O, T, S, Z, J, L)
- ✅ 方块旋转、快速下降、直接落地
- ✅ 等级系统，速度随等级提升
- ✅ 得分系统，连消奖励更多分数
- ✅ 下一个方块预览
- ✅ 方块落地阴影提示
- ✅ 响应式设计，支持移动设备

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| ← (左箭头) | 向左移动 |
| → (右箭头) | 向右移动 |
| ↓ (下箭头) | 加速下降 |
| ↑ (上箭头) | 旋转方块 |
| 空格键 | 直接落到底部 |
| P | 暂停/继续 |

## 🚀 本地运行

直接在浏览器中打开 `index.html` 文件即可：

```bash
# 方式1：直接双击打开 index.html

# 方式2：使用本地服务器
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```
